# fITness

Sledujte svou kondici pomocí naší snadno použitelné fitness aplikace.

## Featury:
- Graf a historie vaší váhy
- Nastavení denních spálených kalorií
- Záznam denních cvičení
- Záznam progresu (v podobě fotek)

## Návod k instalaci
- Přepište soubor .env.example na .env ve tvaru:
 - MONGO_URI=mongodb://user:passwd@host:port/other-options
 - SESSION_SECRET=your_secret_key
- Ve stejném adresáři spusťte v *cmd* příkaz: ```npm install```
